
# Sunkar Finance — Static Site Package

## Что внутри
- `index.html` — Главная страница (RU)
- `kz/index.html` — Базовая казахская версия
- `agent.html` — Страница "Стать агентом"
- `style.css` — Стили
- `script.js` — JS-логика (калькулятор, модальные окна, отправка WhatsApp)
- `assets/images/` — Папка с логотипами, партнёрами и картой

## Как опубликовать на GitHub Pages
1. Распакуйте архив.
2. Создайте репозиторий (или используйте существующий).
3. Закиньте файлы (в корень репозитория или ветку `gh-pages`).
4. В Settings → Pages выберите ветку `main` и папку `/ (root)`.
5. Подождите пару минут — сайт будет доступен.

## Заменить изображения
Просто замените файлы в `assets/images/` (с теми же именами), затем запушьте в репозиторий.
